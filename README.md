# light-fps-operator-demo-godot

Godot 4 migration of the Unity `light-fps-operator-demo`.

## Scope

- Hub scene with operator shop, inventory, and ordered loadout
- Range scene with FPS movement, hitscan shooting, and resettable targets
- Operator pipeline implemented in GDScript
- Global state managed through an autoload singleton

## Run

1. Open the project in Godot 4.
2. Run the default scene.
3. Buy operators in the hub.
4. Add operators to the loadout.
5. Enter the range and test the chain.
